# Vestia — Site beta testeurs

Site vitrine statique pour les beta testeurs de l'application **Vestia** (styliste IA personnel).

- Présentation de l'application et de ses fonctionnalités
- Notes de version (releases)
- Lien vers le formulaire de remontée de bugs / suggestions

## Hébergement

Hébergé via **GitHub Pages** → https://jorishqs.github.io/vestia-site/

Site mono-fichier : `index.html` (HTML + Tailwind CDN, aucun build).

## Mettre à jour les releases

Dans `index.html`, section `<!-- RELEASE TEMPLATE -->`, copiez un bloc `<li>`
et modifiez le numéro de version, la date et le contenu.
